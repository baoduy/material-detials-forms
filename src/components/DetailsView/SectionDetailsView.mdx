# Section Details View

DetialsView is a component allow you render the detials information which a few simple configuration

## Props

<Props of={DetailsView} />

## Usage

Let's say if we have a user json as below. How could we create a Details View with minimal configuration.

<ReactJson src={user} />

### Default Style

<Playground>
  <DetailsView
    labelAlign="left"
    data={user}
    fields={[
      {
        title: 'User Info',
        fields: [
          {
            name: 'name',
            valueOptions: { bold: true },
            gridSize: { md: 6 }
          },
          { name: 'username', gridSize: { md: 6 } },
          'email',

          'website'
        ]
      },
      {
        title: 'Compay Info',
        fields: [
          {
            name: 'name',
            accessor: i => i.company.name,
            valueOptions: { bold: true }
          },
          {
            name: 'Catch Phrase',
            accessor: i => i.company.catchPhrase
          },
          {
            name: 'Bs',
            accessor: i => i.company.bs
          }
        ]
      }
    ]}
    header={{
      title: 'Product Infomation',

        'This will help you to generate multi detials view from single data object with less configuration',

    }}

      <div style={{ textAlign: 'right' }}>
        <Button variant="contained" color="primary">

  />
</Playground>
